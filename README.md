## Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/elhadji-mamadou-sarr/digital_banking_api.git

2. Accédez au répertoire du projet :
    cd digital_banking_api

3. Installez les dépendances :
    pip install -r requirements.txt
   
4. Executer les migrations
   python manage.py migrate

5. Lancer le serveur de developpement
   python manage.py runserver [PORT]
   ## Vous pouvez indiquer le port a utiliser ou utiliser le port par defaut le 8080
